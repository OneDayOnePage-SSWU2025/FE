## :books: 하루한장이란?
하루한장은
함께 책을 읽고 기록하는 독서 그룹 기반 서비스입니다.

<br/>

“혼자서는 꾸준히 읽기 어려운 사람을 위해” </br>
“독서 습관을 만들고 싶은 사람을 위해” </br>
“책을 매개로 서로 연결되고 싶은 사람을 위해”

<br/>

하루한장은
독서 그룹과 기록 기능을 통해
나의 독서 습관을 확인하고, 성취를 시각화하며,
지속 가능한 독서 경험을 제공하는 것을 목표로 합니다.

<br/>

## 주요 기능

### 📖 독서 그룹

독서 그룹 참여
독서 그룹에 참여해 같은 책을 함께 읽고,
서로의 독서 기록을 공유하며 독서 동기를 유지할 수 있습니다.

그룹별 독서 기록 관리
그룹 단위로 읽고 있는 책과 진행 상황을 확인하고,
독서 흐름을 체계적으로 관리할 수 있습니다.

<br/>

### 📝 독서 기록

- 독서 메모 작성
책을 읽으며 느낀 점이나 인상 깊은 내용을 메모로 기록할 수 있습니다.

- 기록 공유
작성한 독서 메모를 그룹 내에서 공유하여
다양한 관점의 독서 경험을 나눌 수 있습니다.

<br/>

### 📊 독서 리포트

- 독서 현황 시각화
읽은 책 수, 페이지 수 등 나의 독서 데이터를 시각적으로 제공하여
독서 습관과 성취도를 한눈에 확인할 수 있습니다.

- 누적 독서 기록 확인
나의 독서 기록을 누적 데이터로 관리하여
장기적인 독서 흐름을 파악할 수 있습니다.

<br/>

### 🐾 캐릭터 성장

- 성장형 캐릭터 시스템
꾸준한 독서를 통해 캐릭터가 변화하며
독서 성취를 시각적으로 표현하고 학습 동기를 부여합니다.
